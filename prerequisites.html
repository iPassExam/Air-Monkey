Before you start your going to need to install the following

 

Ruby 1.9.2

Static-CMS depends on Ruby 1.92.

To check if your system already has ruby installed, type the following on the command prompt:


    	ruby -v

If all went well, ruby will output the version to the console. If ruby is not installed you will receive the error "ruby is not recognized as an internal or external command". So go ahead and install ruby.

Windows users can get it from http://rubyinstaller.org/ (check options to add ruby to your path and associate .rb files)

Abobe Air

If your using the Air-Monkey GUI, your're going to need to install Adobe Air.
